How to use

1. setup database connection in include/dbconfig.php file 
2. import country.sql file OR your own database.
3. run index.php file

Features:
1. Cache Javascript and CSS.
2. Add new records in offline mode and sync when internet connection available.
3. Offline image upload as Data-URI

Roadmap:
In next versions, we aim to cover the following -
1. Update and edit data
2. Sync Data-URI
3. Backup javascript local storage
